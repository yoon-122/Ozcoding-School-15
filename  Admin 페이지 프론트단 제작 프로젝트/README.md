# Admin 페이지 프론트엔드 프로젝트

Bootstrap을 활용한 관리자 페이지 메인 화면 및 회원가입 폼 프로젝트입니다.

## 프로젝트 소개

- 관리자 페이지 메인 화면 제작
- 회원가입 폼 제작

## 주요 기능

### 기본 기능
- 카테고리별 상품 필터링 (상의, 하의, 신발, 패션잡화)
- 제품명 검색 기능
- 상품 목록 테이블 표시
- 페이지네이션 기능

### 챌린지 기능
- 실시간 날짜/시간 표시
- 성별 선택 셀렉터
- 다크모드 토글 기능
- 회원가입 모달 (유효성 검사 포함)
- 회원가입 완료 페이지

## 파일 구조

```
admin-page-project/
├── index.html              # 메인 페이지
├── signup-complete.html    # 회원가입 완료 페이지
└── README.md              # 프로젝트 설명서
```

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 카테고리 선택 및 제품명 검색 기능 사용
3. 다크모드 토글 버튼으로 테마 변경
4. 회원가입 버튼을 클릭하여 회원가입 폼 사용

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.1
